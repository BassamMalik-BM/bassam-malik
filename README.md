# Bassam Malik Crypto Education Website

A complete React + Vite + TypeScript + Tailwind CSS website for beginner-friendly crypto spot trading education.

## Install

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Included

- React Router navigation
- Light/dark theme toggle saved in localStorage
- Mobile hamburger menu
- Responsive premium design
- Framer Motion animations
- Blog listing and blog detail pages
- FAQ accordion
- Risk disclaimer page
- Contact form with React state validation
- Resources cards
